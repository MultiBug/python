```rootsql
14.1 Ввод
# Чтобы получить данные от пользователя, в Python используется функция с интуитивным названием input (англ. ввод).
Например, игра может попросить игрока ввести свое имя и возраст, и использовать их в игре.
Функция input приглашает пользователя ввести данные, после чего введенный текст возвращается в виде строки 
(символы автоматически экранируются).
# Даже если пользователь ввел число, оно воспринимается программой как строка.
```
