```rootsql
66.1 Функция map
# Встроенные функции map и filter - очень полезные функции высшего порядка для работы со списками 
(или с аналогичными объектами, называемыми итерируемыми). 
Функция map принимает функцию и итерируемый объект как свои аргументы и возвращает новый итерируемый объект, 
а функция применяется к каждому аргументу.
def add_five(x):
    return x + 5
nums = [11, 22, 33, 44, 55]
result = list(map(add_five, nums))
print(result)
# Можно добиться того же результата более легким способом с функцией lambda.
nums = [11, 22, 33, 44, 55]
result = list(map(lambda x: x+5, nums))
print(result)
# Чтобы преобразовать результат в список, мы использовали функцию list.
```
```rootsql
66.2 Функция filter
# Функция filter предназначена для фильтрования итерируемого объекта путем удаления элементов, 
которые не соответствуют предикату (функции, которая возвращает логическую переменную).
nums = [11, 22, 33, 44, 55]
res = list(filter(lambda x: x%2==0, nums))
print(res) 
# Как и в случае с map для вывода результата он должен быть вручную преобразован в список.
```
