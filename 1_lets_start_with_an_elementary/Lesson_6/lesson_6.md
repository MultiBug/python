```rootsql
6.1 Неполное частное

# Целочисленное деление осуществляется с помощью двух косых черт (//) и используется для определения частного 
(полного результата деления одного числа на другое).
# Данный код выдает 3, поскольку именно столько раз число 6 помещается в число 20.
print(20 // 6)
# Вы можете осуществлять целочисленное деление и с числами с плавающей точкой (float).
```
