# Кроме сложения, вычитания, умножения и деления Python также поддерживает возведение в степень, 
то есть многократное умножение числа на себя. Эта операция записывается с помощью двух звездочек.
# Вы можете объединить экспоненты вместе. Другими словами, вы можете увеличить число до нескольких степеней. 
Например, 2**3**2.

# Возводить в степень можно и числа с плавающей точкой.

print(9 ** (1/2))
# Имейте в виду, что в результате вы получите число с плавающей точкой.
