Задача 54.3 'Управление складом': 

Словари можно использовать для хранения пар ключ : значение.
Вам необходимо создать программу складского учета для магазина. 
Вы используете словарь для отслеживания всех товаров магазина, а также того, 
какое количество единиц каждого товара имеется в магазине.
store = {"Orange": 2, "Watermelon": 0, "Apple": 8, "Banana": 42}
Дополните предоставленный код, чтобы вывести количество яблок (apple) в магазине.

Подсказка:

Словари можно индексировать точно так же, как и списки, при помощи квадратных скобок.
