```rootsql
25.2 Операции со списками 
# Как и над строками, над списками можно проводить операции сложения и умножения.
# Списки и строки во многом похожи: строки можно рассматривать как списки символов, которые не могут быть изменены.
# Например, строка "Hello", может рассматриваться как список, где каждый символ - элемент списка. 
Первый элемент это "H", второй - "e", и так далее.
```
```rootsql
25.3 Операции со списками
# Когда нужно проверить наличие в списке какого-либо элемента, используется оператор in. 
Ему присваивается значение True (истина), если был найден один символ (или более одного) и False (ложь), если ни одного.
# Оператор in также позволяет определить, является ли строка подстрокой другой строки.
```
```rootsql
25.4 Операции со списками 
# Чтобы проверить действительно ли элемент не находится в списке, используется оператор not одним из следующих способов:
nums = [1, 2, 3]
print(not 4 in nums)
print(4 not in nums)
print(not 3 in nums)
print(3 not in nums)
```
