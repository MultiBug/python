```rootsql
19.1 Логические выражения 
# Еще одним типом данных в Python является логический тип (boolean). Логические выражения принимают только два значения: 
True (истина) и False (ложь).
Они используются для сравнения значений с помощью операторов равенства ==.
# Будьте осторожны, не путайте операции присвоения (один знак равно) и сравнения (два знака равно).
```
```rootsql
19.2 Операции сравнения
# Еще одна операция сравнения (оператор неравенства, !=) возвращает значение True (истина), 
если сравниваемые аргументы неравные и False (ложь), если они равные.
```
```rootsql
19.3 Операции сравнения 
# В Python есть также операции, которые определяют, является ли число (целое или дробное) больше или меньше другого числа. 
Им соответствуют операторы > и <.
# Значения разных типов тоже могут сравнены, например int и float!
```
```rootsql
19.4 Операции сравнения
# Используются также операции больше или равно и меньше или равно: >= и <=.
Они похожи на операторы больше и меньше с той разницей, что они возвращают значение True (истина) при сравнении равных чисел.
# Операторы больше и меньше также используются для сравнения строк в лексикографическом порядке 
(порядок слов основан на алфавитном порядке их букв).
# Вначале сравниваются первые два символа строк "Annie" и "Andy" ("A" и "A"). 
Поскольку они ровны, сравниваются следующие два символа. Так как они тоже ровны, сравниваются третья буква каждой строки 
("n" и "d"). Так как "n" имеет большее алфавитное значение, чем "b", "Annie" больше чем "Andy".
```
