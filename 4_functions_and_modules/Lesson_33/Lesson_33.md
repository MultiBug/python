33.1 Функции
# В дополнение к предопределенным функциям, вы можете создавать свои собственные функции с помощью инструкции def.
Приведем пример функции с именем my_func: она не принимает ни одного аргумента и выводит три раза слово «spam». 
Сначала функция определяется, затем вызывается. Инструкции внутри функции выполняются только при ее вызове.
